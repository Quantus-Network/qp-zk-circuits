# qp-zk-circuits-common

Shared utilities and types for Quantus zk-circuits (Plonky2-based).

Provides field and curve types (`C`, `D`, `F`), codec helpers, circuit gadgets, storage proof handling, and digest utilities used across the Wormhole and other circuit crates. Supports `no_std` when built with `default-features = false`.
