# qp-wormhole-circuit

Wormhole ZK circuit definition (Plonky2).

Defines the circuit logic, targets, and input types for the Wormhole zero-knowledge proof. Exposes `CircuitInputs`, `PrivateCircuitInputs`, and re-exports public input types from [qp-wormhole-inputs]. Used by [qp-wormhole-prover], [qp-wormhole-verifier], and [qp-wormhole-aggregator].
