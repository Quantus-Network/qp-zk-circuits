# qp-wormhole-inputs

Public input types for Wormhole circuit proofs.

Defines the data structures used as public inputs and aggregated outputs for the Wormhole ZK circuit (e.g. `PublicCircuitInputs`, `PublicInputsByAccount`, `BlockData`, `PrivateBatchPublicInputs`). Used by the Wormhole circuit, prover, verifier, and aggregator crates.
